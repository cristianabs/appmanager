THRADEX FOLLOWING THE INDUSTRY

TECHNOLOGICAL INTEGRATION

The integration process involves using the Transport API (TAPI) for northbound interfaces, which allows for efficient and standardized communication between AppManager® and SDN controllers or network management systems. Additionally, the integration utilizes NETCONF/OpenConfig and gNMI/OpenConfig for southbound interfaces, enabling robust and flexible management of network devices.

BENEFITS OF THE INTEGRATION

By integrating AppManager® into current network environments, there are several key benefits to achieve:

1. Enhanced Network Visibility: AppManager® provides comprehensive network inventory and topology visualization, allowing Telefonica to gain deeper insights into its network infrastructure.

2. Improved Efficiency: The use of standardized interfaces ensures streamlined operations and reduces the complexity of network management tasks.

3. Future-Proofing: This collaboration sets the stage for future advancements in network management, as both companies continue to innovate and adapt to emerging technologies.

LOOKING AHEAD

AppManager® by Thradex is a lightweight yet powerful Operations Support System (OSS) designed to enhance operational efficiency for telecom operators. It integrates seamlessly with various network devices and protocols, offering advanced features such as:

• Physical and Logical Inventory Management: Tracks and manages network hardware resources.
• Optical Network Topology Visualization: Provides a clear view of the network’s optical topology.
• Optical Network Management: It supports the management of optical networks, including topology, spectrum, and planning reports, ensuring efficient spectrum utilization and route analysis

AppManager® supports interoperability with industry leaders like Cisco, ZTE, Infinera, Huawei, and Ciena, empowering telecom operators to optimize their network resources and deliver high-quality services.
